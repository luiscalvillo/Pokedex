## Pokedex
A searchable iOS app to find information about Pokemon. 

## Preview
![me](https://github.com/luiscalvillo/Pokedex/blob/main/Screenshots/pokedex-iphone.gif)

## Technologies
* SwiftUI
* MVVM
* PokeAPI

## Features
* All Generation 1 Pokemon
* Pokemon Artwork
* Searchable by name

## Requirements
* iOS 17+
* Swift 5
